# CosmoAI

## AI-Powered Explainable Cosmetic Product Classification and Recommendation Framework

### Overview
CosmoAI is an intelligent cosmetic analytics framework developed using supervised machine learning techniques for cosmetic product classification, suitability assessment, and personalized recommendation.

## Objectives
- Cosmetic Product Classification
- Recommendation System
- Feature Engineering
- Explainable AI
- Ensemble Learning Analysis

## Dataset Information
Target Variable:
- Product Type

Input Features:
- Brand
- Price
- Rating
- Skin Type
- Ingredients
- Premium Brand Indicator
- Natural Ingredient Flag
- Cosmetic Suitability Score

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- SVM
- Naive Bayes
- Gradient Boosting
- Extra Trees

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- SHAP

## Results
Best Performing Model:
Gradient Boosting

Current Accuracy:
~70%
